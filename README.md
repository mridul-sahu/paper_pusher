# Paper Pusher: The Quest for Distributed Mastery 🚀

Welcome to **Paper Pusher**, a research repository and educational hub dedicated to decoding the architectural complexities of **Distributed Machine Learning Systems**.

This project serves as a curated journey through the breakthroughs that power today's largest AI models—focusing on math, orchestration, and the pure engineering of scale.

---

## 📚 Active Research Curriculum

The following papers are currently available in the repository for deep study:

### Scaling Orchestration & Dataflow
*How we bridge the gap between a single chip and a massive cluster.*
*   **[Pathways: Asynchronous Distributed Dataflow for ML](papers/pathways%20paper.pdf)**: Google's next-gen orchestrator for heterogeneous compute.
    *   *Agent Context:* See the [Extracted Pathways Text](papers/pathways_extracted.txt) for LLM-ready analysis.
*   **[GSPMD: General and Scalable Parallelization for ML Programs](papers/gspmd.pdf)**: Mastering sharded dataflow graphs and automatic parallelization.
